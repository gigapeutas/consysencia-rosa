## 📖 Sobre o Projeto
**ConSySenci.a Rosa** é um Web App/PWA interativo, gamificado e 100% gratuito, criado para ensinar idosos a utilizarem smartphones do zero absoluto. Sem jargões técnicos, sem anúncios e com muita empatia. 

O sistema utiliza simulações hiper-realistas de interfaces de celular (começando pelo Android 14 Go Edition) combinadas com uma Assistente de Voz Integrada (A "Rosa"), que guia o usuário passo a passo com analogias do mundo real.

## 🎯 Nosso Propósito
* **100% Gratuito:** O conhecimento básico digital é um direito, não um produto.
* **Zero Anúncios:** Para evitar cliques acidentais e confusão visual.
* **Foco em Segurança:** Ensinar a identificar golpes antes mesmo de ensinar a usar redes sociais.
* **Transferência de Treinamento:** Simuladores exatos aos sistemas reais para criar memória visual real no usuário.

## 📚 Estrutura Pedagógica (Módulos)

* **Módulo 0: A Caixa Mágica (O Físico)**
  * Desmistificando o aparelho (calor, botões e carregamento saudável).
* **Módulo 1: A Escola de Gestos (Ergonomia)**
  * O Toque de Pluma, a Rolagem ("o jornal") e o Toque Longo ("o pregador").
* **Módulo 2: O Teclado Fantasma**
  * Entendendo o teclado que sobe e desce e seus botões de confirmação.
* **Módulo 3: Dicionário de Afetos & Comunicação**
  * Traduzindo Emojis, GIFs e Memes. WhatsApp realista: áudios e fotos.
* **Módulo 4: Escudo de Prata (Segurança)**
  * O Pix seguro, identificação de links falsos e simulação de perfis fakes.

## 🛠️ Stack Tecnológica
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (focado em acessibilidade extrema).
* **Voz:** Web Speech API (Síntese de voz nativa do navegador).
* **Armazenamento:** LocalStorage e Supabase.
* **Hospedagem:** GitHub Pages / Netlify (PWA ready).
